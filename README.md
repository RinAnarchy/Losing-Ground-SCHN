# Losing Ground (失守) - 简体中文汉化补丁

这是一个针对 **Carrioff** 制作的视觉小说游戏 **《Losing Ground》 (失守)** 的非官方简体中文汉化补丁。

> **注意：** 本仓库PC端仅包含汉化补丁文件。请先前往 itch.io 下载原版游戏（安卓版除外）。

## 安装

### 1. 获取游戏本体
请支持原作者，前往 itch.io 下载最新版游戏：
* **原作者**: Carrioff
* **原版地址**: [https://sometotalloser.itch.io/losing-ground](https://sometotalloser.itch.io/losing-ground)

### 2. 下载汉化补丁
点击右侧的 **[Releases (发行版)](../../releases)** 页面，下载最新的汉化补丁文件或补丁安装包。

### 3. 安装步骤
1.  解压原版游戏压缩包。
2.  找到游戏文件夹内的 `game` 目录。
3.  将下载好的 **汉化补丁文件 (.rpa)** 直接放入 `game` 目录下。
4.  运行游戏，在设置中切换中文。

### 4. 安卓安装步骤
1. 下载apk文件。
2. 安装。
3. 游玩。
